Production

Production is the process of converting inputs into outputs to create goods and services that satisfy human wants.

Production Function

Q = f(L, K)

Where Q = Output, L = Labour, K = Capital. The function shows the maximum output achievable from given inputs.

Short Run vs Long Run

⏱ Short Run
  • At least one input is fixed
  • Variable inputs + Fixed inputs
  • Firms can change output only by changing variable inputs
  • Law of Diminishing Marginal Returns applies
⏰ Long Run
  • All inputs are variable
  • No fixed inputs
  • Firms can change scale of production
  • Returns to Scale apply

Total, Average & Marginal Product

📊
Total Product (TP)
Total output produced by all inputs
📈
Average Product (AP)
AP = TP / L
Output per unit of labour
📉
Marginal Product (MP)
MP = ΔTP / ΔL
Extra output from one more unit of labour
AP = TP / L   |   MP = ΔTP / ΔL

Law of Diminishing Marginal Returns

ℹ️
Law of Diminishing Marginal Returns
As more units of a variable input are added to a fixed input, beyond some point the marginal product decreases.

Returns to Scale

↗️
Increasing Returns
Output grows more than proportionally to inputs
→️
Constant Returns
Output grows proportionally to inputs
↘️
Decreasing Returns
Output grows less than proportionally to inputs

Cost of Production

Opportunity Cost

Opportunity Cost = Direct Cost (Explicit) + Indirect Cost (Implicit)

Accounting vs Economic Profit

💵 Accounting Profit
  • TR − Direct Cost (Explicit)
  • What appears in financial statements
💰 Economic Profit
  • TR − (Direct + Indirect Cost)
  • Includes opportunity cost
Accounting Profit = TR − Direct Cost   |   Economic Profit = TR − (Direct + Indirect Cost)

Short Run Costs

💸
TFC
Total Fixed Cost
💹
TVC
Total Variable Cost
💲
TC
Total Cost = TFC + TVC
📉
AFC
AFC = TFC / Q
📈
AVC
AVC = TVC / Q
📊
ATC
ATC = TC / Q
📇
MC
MC = ΔTC / ΔQ
TC = TFC + TVC   |   AFC = TFC/Q   |   AVC = TVC/Q   |   ATC = TC/Q   |   MC = ΔTC/ΔQ

U-Shaped Cost Curves

  Cost
    |
    |   MC
    |  /\
    | /  \   ATC
    |/    \ /\
    |      \/  \    AVC
    |      /\   \
    |     /  \   \
    |    /    \   \
    |   /      \   \
    |  /        \   \______ AFC
    | /          \
    +--------------------- Q

MC intersects ATC and AVC at their minimum points. AFC declines continuously.

Long Run Costs

↗️ Economies of Scale
  • LRAC falls as output increases
  • Specialization, bulk buying, technical efficiency
  • Lower average cost
↘️ Diseconomies of Scale
  • LRAC rises as output increases
  • Coordination problems, bureaucracy
  • Higher average cost

    |   /               \
    |  /                 \
    | /   Economies of    \ Diseconomies
    |/      Scale          \ of Scale
    +------------------------ Q

Revenue

Revenue is the income a firm earns from selling its goods or services.

💵
Total Revenue (TR)
TR = P × Q
📈
Average Revenue (AR)
AR = TR / Q = P
📇
Marginal Revenue (MR)
MR = ΔTR / ΔQ
TR = P × Q   |   AR = TR / Q   |   MR = ΔTR / ΔQ
🏆

Profit

Profit = TR − TC

Optimum Output

Profit is maximized where MC = MR (with MC rising).

Types of Profit

🟩
Super Normal Profit
AC < AR ⇒ P > Cost
🟨
Normal Profit
AC = AR ⇒ P = Cost
🔴
Sub Normal Profit (Loss)
AC > AR ⇒ P < Cost
⛔
Shutdown Condition
AVC = AR ⇒ Close if AR < AVC

Profit Conditions Summary

  Condition              Result
  --------------------------------
  AC < AR (P > ATC)    Super Normal Profit  ✅
  AC = AR (P = ATC)     Normal Profit        🔄
  AC > AR (P < ATC)    Sub Normal Profit     ❌
  AVC = AR (P = AVC)    Shutdown Point       ⛔
  AR < AVC              Shutdown             🚫

Perfect Competition vs Monopoly

🌍 Perfect Competition
  • Many firms (large number)
  • Homogeneous (identical) products
  • Price taker
  • Free entry and exit
  • Zero economic profit in LR
  • P = MC = MR (allocative efficiency)
  • No advertising needed
🔒 Monopoly
  • Single firm
  • Unique product (no close substitutes)
  • Price maker
  • Blocked entry (barriers)
  • Can earn super normal profit in LR
  • P > MC (inefficiency)
  • Can advertise to maintain demand

Short Run vs Long Run in Perfect Competition

⏱ Short Run
  • Fixed number of firms
  • Can earn super normal profit
  • Can earn sub normal profit (loss)
  • Firms cannot exit/enter
⏰ Long Run
  • Firms can enter and exit
  • Super normal profit attracts new firms
  • Losses force firms to exit
  • Equilibrium: Normal Profit (P = ATC)
