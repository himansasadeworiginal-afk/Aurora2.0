# NEXO 1.0

Turn books and notes into interactive web pages.

## Overview

Nexo transforms book summaries and study notes into self-contained, interactive HTML pages.  
Each book and lesson is a fully interactive page with search, expandable sections, and visual tools вЂ” making review and study more engaging than static notes.

## Features

- **Interactive Library Hub** вЂ” Browse all 31 resources with filterable categories and search
- **Category System** вЂ” Books organized into Productivity, Philosophy, Psychology, Power, and Finance; Study Notes organized into Economics and Business Studies
- **Dual Layouts** вЂ” Switch between grid and list view
- **Interactive Summaries** вЂ” Each page includes search, expandable sections, and interactive tools
- **Ambient Environment** вЂ” Particle network, floating orbs, mouse-reactive glow, animated gradients
- **Dark Theme** вЂ” Green/teal/purple/amber color palette with smooth animations throughout

## Usage

Open `nexo 1.0/index.html` in any browser.  
Click a card to open its interactive summary page. Use the category filters and search bar to find specific resources.
